Installation Instructions¶

There are three basic steps when installing CPyMAD:

  • install dependencies
  • build MAD-X as a library
  • build the CPyMad binding

Depending on your platform, this can be more or less complicated.

  • Unix
    • Install dependencies
    • Install libmadx
    • Install CPyMAD
  • Windows
    • Python(x,y)
    • WinPython
    • Other distributions
  • Troubleshooting
    • ImportError: libmadx.so
    • OSError: Missing source file
    • Unable to find vcvarsall.bat
    • unrecognized command line option ‘-mno-cygwin’
