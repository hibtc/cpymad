Dependencies¶

Note

Before installing any dependencies, take a look at the OS specific install recommendations! These contain remarks about the preferred install method.

To build MAD-X and CPyMAD from source you will need

  • CMake
  • python>=2.6
  • C / Fortran compilers.

Furthermore, CPyMAD depends on the following python packages:

  • setuptools
  • Cython
  • NumPy
  • PyYAML

The python packages can be installed using pip.
